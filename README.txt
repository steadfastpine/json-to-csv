Json to CSV



# Contact: https://www.linkedin.com/in/steadfastpine/

# Release Date: 2021-07-08
# Version: .1



Description

	Convert single layer json data into csv format.



Prerequisites

	Perl



Installation

	Download and unzip the program files, then change working directory to them:
	
		# cd json-to-csv
